Track or stage file a \
`git add fileName`  
\
Stage all tracked file
\
`git add -u`

Unstage all file  
`git reset`  
\
Unstage file  
`git reset fileName`  
\
Reset to HEAD   
`git reset --hard`  
or  
`git reset --hard HEAD`  || `git reset --hard @`  
\
Reset to commit   
`git reset --hard <HASH>` 

Reset to origin  
`git fetch`  
`git reset --hard origin/main`
